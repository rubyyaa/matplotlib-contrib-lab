# Gallery Examples

Each `.py` file in this directory tree is a standalone gallery example.
See [CONTRIBUTING.md](../../CONTRIBUTING.md) for the required file format.

## Subdirectories

- **signals/** -- Signal processing (FFT, spectrogram, discrete-time)
- **circuits/** -- Circuit analysis (transients, phasors, impedance)
- **controls/** -- Control systems (Bode, Nyquist, pole-zero, step response)
- **communications/** -- Digital communications (constellation, eye diagram)
- **sensors/** -- Sensors and measurement (calibration, uncertainty)
